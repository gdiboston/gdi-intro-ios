CocoaPods

  • Manages library dependencies for Xcode projects
  • Dependencies are specified in a single text file called a Podfile
  • Resolves dependencies between libraries, fetches the required source code and links everything together in an Xcode workspace to build your project

Reference:

https://guides.cocoapods.org/using/getting-started.html

Installing CocoaPods

  1. Open Terminal and enter:
    2. sudo gem install cocoapods

NOTE: CocoaPods is built with Ruby and it will be installable with the default Ruby available on OS X

Syntax I

Access a property using a 'dot'

    arm.hand;
    arm.elbow;

Can even chain them together:

    arm.hand.pinkieFinger;

Functions

Functions = actions

Functions can 'return' things:

- (Pen *)giveMeYourPen
{
    return myPen;
}

Functions can accept things (called 'arguments'):


- (void)takeThisPen:(Pen *)thisPen
{
    myPen = thisPen;
}
'=' means call the thing on the right by what's on the left 'Pen' is the type of object 'thisPen' is the name of the object You could name it something other than 'thisPen'. 'Mac'? Gwyneth Paltrow: child named 'Apple.' You can do that, it's just weird (please name things logically!)

Can do both: 


- (Pen *)penWithInkCatridge:(InkCartridge *)cartridge
{
    myPen.inkCartridge = cartridge;
    return myPen;
}

Syntax II

Tell an object to do something with brackets:

    [hand open];
    [hand close];

You can chain these too:

    [[student giveMeYourPen] makeMark];

Or combine them with properties:

    [arm.hand.pinkieFinger wiggle];
